Task 2: Step 3 Contract

POST /api/borrows or /api/borrowing

{
  "studentId": "U1001",
  "bookId": "B2005",
  "borrowDate": "2026-03-10",
  "returnDate": "2026-03-15"
}


java
public interface BorrowService {
    BorrowRecord borrowBook(String userId, String bookId);
}

golang
type BorrowService interface {
    BorrowBook(userID string, bookID string) (*BorrowRecord, error)
}


Expected Response
{
  "borrowId": "BR001",
  "studentId": "U1001",
  "bookId": "B2005",
  "status": "BORROWED",
  "borrowDate": "2026-03-10",
  "dueDate": "2026-03-15"
}



Table Design

กลุ่ม IDENTITY
1) users table
- userId       STRING
- email        STRING
- passwordHash STRING

2) profiles table
- userId      STRING
- displayName STRING
- avatarUrl   STRING

3) userPreferences table
- prefId    STRING
- userId    STRING
- prefKey   STRING
- prefValue STRING


กลุ่ม AUTH
4) userSessions table
- sessionId    STRING
- userId       STRING
- accessToken  STRING
- refreshToken STRING
- issuedAt     DATE
- expiresAt    DATE


กลุ่ม BILLING
5) plans table
- planId        STRING
- name          STRING
- tier          STRING
- price         FLOAT
- billingCycle  STRING
- maxDevices    INT
- maxResolution STRING
- isActive      BOOLEAN

6) subscriptions table
- subscriptionId STRING
- userId         STRING
- planId         STRING
- status         STRING
- startDate      DATE
- endDate        DATE

7) payments table
- paymentId             STRING
- subscriptionId        STRING
- externalTransactionId STRING
- status                STRING
- amount                FLOAT
- currency              STRING
- paidAt                DATE
- deletedAt             DATE


กลุ่ม CONTENT
8) videos table
- contentId  STRING
- title      STRING
- cdnUrl     STRING
- resolution STRING
- duration   INT
- codec      STRING

9) music table
- contentId STRING
- title     STRING
- cdnUrl    STRING
- artist    STRING
- bitrate   INT
- genre     STRING
